# ScoutMe Frontend

## Descripción

ScoutMe es una plataforma de scouting deportivo orientada a futbolistas amateurs. Permite visualizar perfiles deportivos, estadísticas y comparar jugadores para facilitar el contacto entre jugadores, entrenadores y reclutadores.

## Tecnologías utilizadas

* React
* Vite
* React Router DOM
* CSS3

## Instalación

```bash
git clone https://github.com/gonzavadone10/scoutme-frontend.git
cd scoutme-frontend
npm install
```

## Ejecución

```bash
npm run dev
```

La aplicación estará disponible en:

http://localhost:5173

## Arquitectura

src/

* components/
* pages/
* assets/
* App.jsx

## Funcionalidades

* Inicio
* Registro e inicio de sesión
* Listado de jugadores
* Perfil de jugador
* Comparador de jugadores
* Mi perfil deportivo
* Carga de estadísticas
* Contacto
